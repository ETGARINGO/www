👑 Opening an Elevated Prompt

  1. Click Start, type cmd or powershell
  2. Press Ctrl + Shift + Enter (or right-click → "Run as administrator")
  3. Click "Yes" on the UAC prompt
  4. The window title will show "Administrator:" when elevated

⚠️ Commands tagged "admin" or with a red Advanced badge require this elevated prompt. Only run commands you understand — an elevated shell can make system-wide changes.
